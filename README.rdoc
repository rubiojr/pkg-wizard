== Installation

gem install pkg-wizard

=== Requirements

Git >= 1.7

* Debian/Ubuntu

apt-get install build-essential ruby1.8-dev mock rpm yum git-core createrepo

* RHEL/Fedora

yum install rpmdevtools git createrepo mock yum-utils
